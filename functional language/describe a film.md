# Describing a Film - Structure and Useful Phrases

## 1. Describe the Film

- The film is called …
- It is a … (genre: drama / comedy / thriller / fantasy / cartoon, etc.)
- The film is about …
- The main idea/message of the film is …

## 🌍 2. Setting

- The film is set in … (place / country / city)
- The action takes place in … (time: the past / present / future / 19th century, etc.)

## 👤 3. Main Characters

- The main character is …
- He/She is … (describe personality, job, age, qualities)
- Other important characters are …

## 📖 4. Plot (Main Events)

Tell the story in chronological order:

- To begin with, …
- At the beginning of the film, …
- First of all, …
- After a while, …
- Then, …
- Soon after that, …
- Meanwhile, …
- At the same time, …
- Later on, …
- After that, …
- All of a sudden, …
- Suddenly, …
- Unexpectedly, …
- Because of that, …
- That is why, …
- As a result, …
- For this reason, …
- However, …
- Although, …
- In spite of that, …
- Eventually, …
- In the end, …
- Finally, …
